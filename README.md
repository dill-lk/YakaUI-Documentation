**YAKA UI** is a lightweight, high-performance UI framework designed for creating stunning, interactive web interfaces. Version 1.0 is optimized for production, leveraging **GSAP (GreenSock Animation Platform)** for buttery smooth 60fps animations and **SVG** for crisp, scalable graphics.

Built with modern **ES Modules** and **CSS3 Variables**, YAKA UI brings premium "Glassmorphism" aesthetics and complex physics-based interactions to your project without the bloat.

## ğŸ“¦ Installation (v1.0)

Simply include the production-ready `yaka_v1.0.css` and `yaka_v1.0.js` files via CDN.

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/dill-lk/YAKA-UI/yaka_v1.0.css">

<script type="module" src="https://cdn.jsdelivr.net/gh/dill-lk/YAKA-UI/yaka_v1.0.js"></script>

```

> **Note**: YAKA UI v1.0 handles GSAP imports automatically. Ensure you have an internet connection or host the dependencies locally for offline use.

## ğŸ”¨ Usage

YAKA UI automatically initializes core components upon loading. Use specific **data attributes** to activate premium features instantly.

### Basic Template

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/dill-lk/YAKA-UI/yaka_v1.0.css">
</head>
<body>
    
    <button class="btn-primary" data-magnetic>Hover Me</button>

    <script type="module" src="https://cdn.jsdelivr.net/gh/dill-lk/YAKA-UI/yaka_v1.0.js"></script>
</body>
</html>

```
